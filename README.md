# Tea
Reference implementation for the esoteric programming language *[Tea](https://esolangs.org/wiki/Tea).*

## Installation
1.) Download both `tea.py` and `tea.bat` from the `...\src` subdirectory<br>
2.) Choose an installation location and create a subfolder inside of it called `...\tea`<br>
3.) Move `tea.py` and `tea.bat` into that folder<br>
4.) Add the `...\tea` directory to the PATH environment variable

## Usage
In Command Prompt or Powershell, Tea programs can be executed using
```
tea path\to\file.tea
```
