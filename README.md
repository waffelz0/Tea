# Tea
Reference implementation for the esoteric programming language [Tea](https://esolangs.org/wiki/Tea).

## Installation
1.) Download both `tea.py` and `tea.bat`<br>
2.) Create a folder in the directory of your choice called `\tea`<br>
3.) Move `tea.py` and `tea.bat` into that folder<br>
4.) Add the `\tea` directory to the PATH environment variable

## Usage
```
tea path\to\file.tea
```
